# Smart e-KYC Verification System

## Description
Smart e-KYC Verification System is a backend application that automates digital identity verification. Users can register, log in, and upload KYC documents, which are securely stored in MongoDB. Admins can review and approve or reject submissions. Built with Node.js, Express.js, and MongoDB to demonstrate real-world KYC workflows

## Features
- User Registration & Login
- Document Upload
- Admin Verification
- Secure Data Storage

## Technologies Used
- Node.js
- Express.js
- MongoDB
- JavaScript

## How to Run
1. Clone the repository
2. Run npm install
3. Run node app.js
